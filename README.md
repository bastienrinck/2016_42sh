Project 42sh from the end of the first year at EPITECH

# But
Make a complete shell with an LL parser.

# Fonctionnel
- Historique (Up arrow, down arrow, !$, !!, !:n, !n, !-n, ...)
- Job Control (&, jobs [-l ], fg [%n], bg [%n])
- Globbing ([m,a]?*.[a-o])
- Variables locales (set, unset)
- Alias (alias, unalias)
- Édition de ligne avancée
- ~/.42shrc
- Auto-complétion (deux modes: Ctrl + F)
- Magic quote
